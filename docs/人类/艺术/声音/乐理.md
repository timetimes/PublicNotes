# 乐理

乐理、乐谱、乐器

声源—（振动）—介质——受体


|    |   声音的频率   |声音的振幅|
| --- | --- |---|
|定义|   每秒声音振动的次数  |   声音的能量  |
|人的听力范围|20Hz~20,000Hz|20微帕~200,000,000微帕|
|人的感受|频率低，低沉；频率高，高亢|振幅低，声音小；振幅高，声音大|

音量通常用分贝表示

自然界不存在单一频率的声音，任何声音都是一组各种频率的声波的组合
噪音就是频率和音量杂乱无章的声音

乐音=基音+泛音/谐音

泛音--》曲调（由于泛音共振）、和弦


人类比较舒适的声音范围100Hz~4000Hz
人类对音高的感觉是等比关系 
音程100~200；150~300；200~400 听起来是一样的

音符频率的选定：每一个音程分几个音
